# OOP-Chat-System
Mini Chat System Using OOP Concept
OOP Chat System (Python)

A simple chat system built using Object-Oriented Programming concepts in Python.

Features⚡

- Users can join a chatroom
- Users can send messages
- Chat history is stored
- Users can leave the chatroom

OOP Concepts Used

- Classes and Objects
- Constructors (init)
- Class variables
- Object interaction

Classes

- User
- Message
- ChatRoom

Example

Users join a chatroom, send messages, and view the chat history.
